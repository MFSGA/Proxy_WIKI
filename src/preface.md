# 序言

代理应用的生态越来越拥挤且碎片化，因此——出于个人兴趣——我决定把自己在用的内容整合成一个统一栈。
- `chimera_Client` 参考了开源项目 [clash-rs]，长期目标是追平 [Mihomo] 的功能集。
- `Chimera` 借鉴 [clash-nyanpasu] 项目，但最大区别在于我优先支持 `chimera_client` 引擎本身，它可以被视作 `clash-rs` 的核心。
- `Chimera_Server` 项目可以看作 [xray-core] 的 Rust 实现，目标仍是保持与原始 `xray-core` 的兼容性。
